## 2. preprocessing

* Transpose하고 첫번째 row를 column_header로 정하기
```python
def clean(file_name):
  df = pd.read_csv(Url_head + file_name).transpose()                           #.transpose() 대신 .T 써도 됨
  column_head = df.iloc[0]
  df = df[1:]
  df.columns = column_head
return df                                                                    #마지막 row만 return하고 싶으면 df[-1:]
```
