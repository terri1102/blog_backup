# Database Modeling



### 모델링이란?



### 모델링의 특징

1. 추상화(모형화, 가설적)
2. 단순화
3. 명확화



### 데이터 모델링시 유의점

1. 중복
2. 비유연성
3. 비일관성



### 데이터 모델링의 3단계

1. 개념적 데이터 모델링(Conceptual Data Modeling)
2. 논리적 데이터 모델링(Logical Data Modeling)
3. 물리적 데이터 모델링(Physical Data Modeling)
