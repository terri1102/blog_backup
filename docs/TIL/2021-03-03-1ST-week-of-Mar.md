Git 에러 CRLF will be replaced by LF

맥 또는 리눅스를 쓰는 개발자와 윈도우를 쓰는 개발자가 협업할 때 white space 때문에 나는 오류

이 테마를 만든 사람이 맥이나 리눅스를 썼나 보다.

```python
git config --global core.autocrlf true #이 프로젝트에서만 쓸 거면 global 빼기
```
